# Address API Spec

## Create Address

Endpoint : POST /api/contacts/:contactId/addresses

Headers :

- Authorization: token

Request Body :

```json
{
  "street": "Jalan Contoh, optional",
  "city": "Kota, optional",
  "province": "Provinsi, optional",
  "country": "Negara Apa",
  "postal_code": "123123"
}
```

Response Body

```json
{
  "data": {
    "id": 1,
    "street": "Jalan Contoh, optional",
    "city": "Kota, optional",
    "province": "Provinsi, optional",
    "country": "Negara Apa",
    "postal_code": "123123"
  }
}
```
